O sistema é uma criação fictícia de um sistema de gerenciamento de planetas controlado por um usuário administrador. O sistema consiste em o usuário administrador poder logar no sistema com suas credencia controlar seus planetas;

Com base nisso, o usuário administrador poderá realizar as seguintes ações:
Acesso a todos os planetas;
Acesso aos detalhes de cada planeta;
Criar um planeta novo (IMPORTANTE: Ao criar um planeta novo, o mesmo precisa aparecer no fim da listagem);
Filtrar os planetas por nome;
Filtrar os planetas pelo valor de uma característica escolhida;
Editar um planeta;
Excluir um planeta (IMPORTANTE: Durante a exclusão, é preciso perguntar se o usuário tem certeza antes);
Ver comparação geral entre os planetas através de uma tabela

Crie também um arquivo .json que controle o login do usuário.    

OBSERVAÇÕES: Melhorias são bem vindas, implementações extras também. Atente-se apenas em focar antes em entregar o proposto para depois tentar trazer soluções novas. O layout precisa ser inteiramente respeitado, tanto na parte da visualização quanto de ações.
